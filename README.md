# A-few-questions-about-the-World
A small quiz about a big deal.
______________
## Prerequisites:
Python 3.5 or newer

--------------
## Starting:
### on Windows and Mac:
#### Run the application using 'start.py'.
In console:
   ```
python3 start.py
```
### on Linux:
#### Run the application using 'few_questions' file:
Make the file executable and double click on it.
#### ... or 'start.py'
In terminal:
 ```
python3 start.py
```
